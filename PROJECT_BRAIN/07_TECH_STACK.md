# 🛠️ 07_TECH_STACK.md
## Frontend
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4.0
- **Language**: TypeScript
- **UI Components**: React 19, Lucide React (icons)
- **State Management**: React Context / Hooks

## Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (Mongoose ODM)
- **File Storage**: Cloudinary (Product images)
- **Security**: JWT, Bcrypt, Helmet, CORS
- **AI Integration**: Google Gemini AI (Generative content)

## Infrastructure
- **Deployment**: Local dev (scripts/seed), Production ready with Express server.
- **Automation**: Custom scripts for content uploading and product seeding.
